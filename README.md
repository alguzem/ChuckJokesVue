# ChuckJokesVue 

## Description
ChuckJokesVue is a simple web application developed using Vue.js that displays a list of Chuck Norris jokes dynamically.

This project demonstrates the use of Vue directives such as **v-for** for rendering lists and basic data binding.

---

## Objective
The objective of this project is to understand the fundamentals of Vue.js, including:
- Creating a Vue instance
- Managing data dynamically
- Rendering lists using v-for
- Applying basic styling with CSS

---

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Vue.js (CDN)

---

## Features
- Dynamic rendering of jokes using Vue
- Clean and minimalist UI
- Responsive design
- Easy to understand structure
